# LQS Landing Page

LQS(Learning Quality System) 프로젝트의 공식 랜딩 페이지입니다.

## 🌐 Live Demo

https://ksyksy819.github.io/-_-/

## 🚀 기술 스택

- **React** - UI 라이브러리
- **Vite** - 빌드 도구
- **Tailwind CSS** - 스타일링
- **GitHub Pages** - 호스팅

## 📦 설치 및 실행

### 로컬 개발

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

개발 서버는 http://localhost:5173 에서 실행됩니다.

### 빌드

```bash
# 프로덕션 빌드
npm run build

# 빌드 미리보기
npm run preview
```

## 🎨 페이지 구조

1. **Hero** - 첫 인상과 핵심 가치 전달
2. **Problem Statement** - 해결하려는 문제 정의
3. **Solution Overview** - LQS 솔루션 소개
4. **Features** - 6가지 핵심 기능
5. **User Personas** - 3가지 타겟 사용자
6. **How It Works** - 4단계 사용 흐름
7. **Tech Stack** - 기술 스택 소개
8. **Roadmap** - 프로젝트 로드맵
9. **CTA** - 행동 유도
10. **Footer** - 푸터 정보

## 🔄 자동 배포

`main` 브랜치에 push하면 GitHub Actions를 통해 자동으로 GitHub Pages에 배포됩니다.
