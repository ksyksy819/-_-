## 2. Problem Statement

### 섹션 헤딩
```
왜 같은 유형을 반복해서 틀릴까요?
```

### 리드 텍스트
```
정답률만 보여주는 기존 시스템은 
'왜 틀렸는지', '어떤 생각으로 답했는지'를 알려주지 못합니다.
```

### Pain Point 1
**제목**: 사고 과정이 보이지 않음
**설명**: 
```
문제풀이 결과는 O, X로 남지만
학생이 어떤 생각으로 답을 선택했는지는
데이터로 수집되지 않습니다.
```

### Pain Point 2
**제목**: 질문 습관을 추적할 수 없음
**설명**: 
```
"좋은 질문을 하라"고 말하지만
질문의 질이 성장하는지 확인할 방법이 없습니다.
```

### Pain Point 3
**제목**: 회고가 지속되지 않음
**설명**: 
```
회고의 중요성은 알지만
매일 수동으로 작성하기엔 너무 귀찮고
결국 며칠 만에 포기하게 됩니다.
```

## 3. Solution Overview

### 섹션 헤딩
```
LQS는 이렇게 해결합니다
```

### Value Proposition
```
우리는 진지하게 성적을 올리고 싶은 수험생과 이들을 가르치는 학원·교사가
문제풀이·노션·LMS·ChatGPT를 다 써봐도 
'왜 같은 유형을 반복해서 틀리는지',
'학생이 어떤 생각으로 답을 만들었는지' 보이지 않을 때,

사고 과정·질문 흔적을 자동으로 수집·분석하고
메타인지·질문력을 시각화하여 성장하도록 돕습니다.
```

### 핵심 차별점

**1. 사고 과정 자동 수집**
```
문제를 풀 때의 생각을 자연어 또는 선택형으로 간편하게 입력.
AI가 자동으로 구조화하고 분석합니다.
```

**2. AI 기반 메타인지 점수화**
```
사고 단계, 오류 유형, 근본 원인을 자동 분류하고
메타인지 수준을 점수화하여 성장을 추적합니다.
```

**3. 질문력 성장 시각화**
```
생성한 모든 질문을 분석하여
질문의 깊이, 다양성, 비판적 사고 수준을 측정하고
시간에 따른 변화를 그래프로 보여줍니다.
```

## 4. Key Features

### Feature 1: 사고흐름 캡처
**제목**: 생각의 흐름을 놓치지 마세요
**설명**:
```
문제를 풀면서 떠오르는 생각을 자유롭게 기록하거나,
빠른 선택지로 사고 유형을 표시하세요.
AI가 자동으로 구조화하여 분석합니다.
```
**주요 기능**:
- 자유 서술형 입력
- 빠른 선택형 입력 (근거 비교, 조건 해석 등)
- 실시간 AI 분석

### Feature 2: AI 사고 분석
**제목**: AI가 당신의 사고를 분석합니다
**설명**:
```
OpenAI GPT 기반 분석 엔진이
사고 단계를 분류하고, 오류 유형을 식별하며,
근본 원인을 추출합니다.
```
**주요 기능**:
- 사고 단계 분류 (이해→해석→비교→추론→결정)
- 오류 유형 식별 (개념 부족, 조건 해석 미흡 등)
- 메타인지 점수 자동 계산

### Feature 3: 오답 이유 분석
**제목**: 왜 틀렸는지 명확하게
**설명**:
```
단순히 "오답"이 아니라
"왜 그렇게 생각했는지", "무엇이 부족했는지"
근본 원인을 자동으로 추출합니다.
```
**주요 기능**:
- 오답 이유 자동 분류
- 유사 오답 패턴 클러스터링
- 개선 질문 자동 생성

### Feature 4: 질문력 분석
**제목**: 질문하는 능력도 성장합니다
**설명**:
```
생성한 모든 질문을 분석하여
유형, 깊이, 구체성, 확장성을 측정하고
질문 역량의 질적 성장을 시각화합니다.
```
**주요 기능**:
- 질문 유형 분류
- 질문 깊이 점수화
- 질문 다양성 그래프

### Feature 5: 자동 회고 생성
**제목**: 귀찮은 회고, AI가 대신합니다
**설명**:
```
오늘 푼 문제와 사고 분석 결과를 바탕으로
AI가 자동으로 의미 있는 회고문을 생성합니다.
입력 부담은 0, 효과는 100.
```
**주요 기능**:
- 매일 자동 회고 생성
- 사고 취약점 요약
- 내일의 집중 포인트 제시

### Feature 6: 성장 대시보드
**제목**: 나의 성장을 한눈에
**설명**:
```
메타인지 점수, 질문력 점수, 사고 패턴 변화를
시각적 대시보드로 확인하세요.
30일간의 변화를 히트맵으로 보여줍니다.
```
**주요 기능**:
- 메타인지/질문력 점수 추적
- 월간 사고 패턴 분석
- 사고 실패 유형 비중 차트

## 5. User Personas

### 페르소나 1: 수험생 - 김민하
**인용구**:
```
"같은 유형을 또 틀렸어요. 
제 사고 패턴에 뭔가 문제가 있는 것 같은데,
정확히 뭔지 모르겠어요."
```

**핵심 Needs**:
- ✅ 사고 흐름 자동 분석
- ✅ 오답 원인 자동 추출
- ✅ 질문 만들기 도움
- ✅ 내 사고 패턴 변화 추적

**LQS가 제공하는 가치**:
```
매 문제마다 생각을 기록하고, AI 분석을 통해
자신도 몰랐던 사고 습관을 발견할 수 있습니다.
```

### 페르소나 2: 재도전생 - 박지윤
**인용구**:
```
"회고가 중요한 건 알지만, 
매일 노션에 정리하기엔 너무 피곤해요.
자동으로 정리돼주면 좋겠어요."
```

**핵심 Needs**:
- ✅ 회고 자동 생성 (부담↓)
- ✅ 사고·질문 패턴 변화 그래프
- ✅ 오답 이유 자동 분류
- ✅ 학습 전략 자동 추천

**LQS가 제공하는 가치**:
```
입력 부담 없이 자동으로 생성되는 회고와
데이터 기반 학습 전략으로 효율적인 재도전을 돕습니다.
```

### 페르소나 3: 학원 원장 - 최성훈
**인용구**:
```
"학생들의 사고력을 키우고 싶지만,
선생님들에게 추가 업무를 줄 수는 없어요.
자동화된 시스템이 필요합니다."
```

**핵심 Needs**:
- ✅ 학생 사고·질문 데이터 자동 수집
- ✅ 반/학생별 사고력·질문력 대시보드
- ✅ 학부모 상담용 사고력 리포트
- ✅ 교사 업무 증가 없는 자동화

**LQS가 제공하는 가치**:
```
교사의 추가 업무 없이 학생 데이터를 자동 수집하고,
학부모 상담에 활용할 수 있는 리포트를 제공합니다.
```

## 6. How It Works

### 섹션 헤딩
```
4단계로 간단하게 시작하세요
```

### Step 1
**제목**: 문제 풀이 + 사고 입력
**설명**:
```
문제를 풀면서 떠오르는 생각을 간단하게 입력하세요.
자유 서술 또는 빠른 선택지 중 편한 방식으로.
```

### Step 2
**제목**: AI 실시간 분석
**설명**:
```
OpenAI 기반 분석 엔진이 즉시 사고 과정을 분석하고
구조화된 데이터로 변환합니다.
```

### Step 3
**제목**: 오답 패턴 자동 분류
**설명**:
```
틀린 문제의 근본 원인을 자동으로 추출하고
유사한 실수를 클러스터링합니다.
```

### Step 4
**제목**: 성장 리포트 확인
**설명**:
```
대시보드에서 메타인지 점수, 질문력 변화,
사고 패턴 히트맵을 확인하세요.
```

## 8. Roadmap

### 섹션 헤딩
```
프로젝트 로드맵
```

### MVP (6-8주) - 현재 진행 중
**목표**: 핵심 가치 검증
**기능**:
- ✅ 문제풀이 + 사고흐름 입력
- ✅ AI 기반 사고 분석
- ✅ 오답 유형 자동 분류
- ✅ 질문 분석 + 질문력 점수
- ✅ 일일 자동 회고
- ✅ 개인 대시보드

### V1 확장 (8-12주)
**목표**: B2B 확장
**기능**:
- 🔄 학원/교사용 대시보드
- 🔄 학생별 비교 분석
- 🔄 학부모 상담 리포트

### V2 확장 (12주+)
**목표**: 고도화 및 자동화
**기능**:
- 📅 반별/유형별 AI 학습전략 추천
- 📅 풀이 과정 자동 수집 (화면/필기 기반)
- 📅 음성 기반 사고흐름 입력
- 📅 학교용 LTI 연동
