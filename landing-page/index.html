3단계로 간단하게

기존 시험지 그대로, AI가 채점을 도와드립니다

📤

1. 업로드

기존 PDF 시험지를 그대로 업로드하세요. 재제작 필요 없습니다.

🤖

2. AI 분석

AI가 자동으로 답칸을 탐지하고 채점 초안을 생성합니다.

✅

3. 확정

교사가 검토하고 승인만 하면 끝. 필요시 수정도 가능합니다.

핵심 기능

교사와 학생 모두를 위한 스마트한 기능들

🎯

AI 답칸 자동 탐지

PDF에서 빈칸, 줄, 표를 자동으로 분석하여 답안 입력 영역을 제안합니다.

✍️

손글씨 인식 채점

학생의 필기 이미지를 분석하여 점수와 근거를 포함한 초안을 생성합니다.

📊

4수준 평가 체계

초등 교육에 맞는 매우잘함/잘함/보통/노력요함 평가 기준을 지원합니다.

📝

누가기록 자동 생성

성취기준과 평가 결과를 바탕으로 생활기록부 문장을 자동 생성합니다.

🔄

실시간 자동 저장

5초 단위로 자동 저장되고, 오프라인에서도 작업이 유지됩니다.

👀

실시간 현황 모니터링

학생들의 접속, 작성 중, 제출 상태를 실시간으로 확인할 수 있습니다.

왜 GradeFlow인가요?

다른 서비스와 차별화되는 핵심 가치

📄

기존 시험지 그대로

이미 만들어둔 PDF 시험지를 재제작 없이 바로 사용할 수 있습니다. 새로운 형식을 배울 필요가 없습니다.

🤝

AI 초안 + 교사 확정

완전 자동이 아닌, AI가 초안을 만들고 교사가 최종 확정합니다. 신뢰성과 효율성 모두 확보합니다.

🔒

철저한 보안

학생별 고유 코드로 자기 페이지만 접근 가능. 타인의 시험지나 성적은 절대 볼 수 없습니다.

📶

오프라인도 안심

인터넷이 끊겨도 작업 내용이 보존됩니다. 재접속 시 자동으로 동기화됩니다.
